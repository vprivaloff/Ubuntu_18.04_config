# Разные полезные команды

+ удалить содержимое каталога linux

    `rm -rf /home/catalog/*`
    
    или
    
    `sudo rm -rf /home/catalog/*`

+ рестарт apache2

    `sudo systemctl restart apache2`

+ рестарт Redis

    `sudo service redis-server restart`

+ рестарт MySQL
    
    `sudo systemctl restart mysql.service`
